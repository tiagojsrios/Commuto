# Commuto

Commuto is a simple tool, targeting MacOS operating systems, that shows when the next recommended commute trip will be.

## Tech Stack

- Swift

*DISCLAIMER: I have never developed apps using Swift. This tool is mostly vibe coded.*

My knowledge of Swift and architecturing apps in Swift is very limited. Though I'm open for suggestions on how to improve the code. Feel free to contribute!

## Can I use it?

Yes! Or better, it depends. 

As a citizen of the Netherlands myself, this tool uses the public API of the Dutch railways company, Nederlandse Spoorwegen (NS). Therefore, if you live outside of the Netherlands, unfortunately you won't be able to use it.

## How to use

I don't have an Apple Developer Account and it isn't in my plans to get one. Therefore, the redistribution of this tool will only happen through the .dmg published together with the release tag.

In order to successfully run the app, you will have to request a free API key through the developer portal of NS.

After installing the app, a new icon should show on your top bar when you first run it. To add your newly created API key, click on the icon and then 'Settings'.

Last, you have to add your commute details - meaning the nearest station to your work and the destination.
